Quick BackTest

The Quick BackTest is shown below:

Description of fields

 

Symbol

Mandatory field.

It is a text field to input the underlying symbol.

 

Expiration

 

Risk $ per Trade

Mandatory field.

 

Mandatory field.

Provides the ability to trade weekly or monthly options.

 

The amount the user is willing to risk per trade.

 
 

Strike

 

Mandatory field.

 

The strike selection for the trade.  In the money (ITM), at the money (ATM), or out of the money (OTM).
 

Strategy

 

Mandatory field.

 

This field provides the list of options strategies that can be used.

 

Days to Expire

 

Mandatory field.

 

This fields requires the user to manually enter the minimum number or days to expiration for the option trade.
 

Profit Target (%)

Mandatory Field.

 

Exit position based on percent of profit made.
 

Ignore Exit Date

Default set to No

 

Does not ignores the exit date when No selected.
 

State

Mandatory Field.

Delphian Proprietary State Modelling™ lets you choose from state 1 through 8.
 

Stop Loss (%)

Mandatory Field.

Provide a specified price in order to limit a loss.

 

Execute a Quick BackTest

To execute the Quick BackTest:

  1. Input data into each required field of the Quick BackTest panel.
  2. Click to execute the test.

  1. When the test is successfully executed, the button appears next to the button. Click the button to view the results.

  1. The summary panel displays the results of the test.

You can populate the same fields as the previously run quick back tests by clicking   from the Last 10 Quick BackTests panel.
