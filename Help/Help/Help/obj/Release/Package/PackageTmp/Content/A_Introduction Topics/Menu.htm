The Home Page of the Delphian platform provides access to the main menu; displays the current markets conditions along with customizable tabs and it has quick-links with useful shortcuts.

The Home page has the following components:

  1. Dashboard - See Dashboard.
  2. Menu - See Menu.
  3. Quick-links - See Dashboard.

 

Dashboard

The Home Page of the Delphian platform, is the dashboard. The dashboard gives you an overview of the market.

This page has tabs to customize the market information.

NOTE: Some of the tabs mentioned below may not be visible to you depending on the mode and platform chosen during subscription.

Here you can:

  • analyze markets with your watch list from the My Watchlist tab to display current market conditions for user added symbol
  • the current market pulse from the Market Pulse tab to display State Modeling™ pie chart of the current market conditions and
  • access your daily watch on all stocks from the DailyWatch that allows user to save signals to scan for stocks on a daily basis.

In the Dashboard we also have Daily Alert and Idea Nook:

  • The Daily Alert is where you will find trade alerts to the backtest models you have previously built. For example, if you have a plan that sells options every Monday, you will see a trade alert in the Daily Alert section for every Monday.
  • The Idea Nook gives trade ideas based on our proprietary buy/sell algorithm, State Modeling.

Menu

The left side of our home screen is the main menu. The Menu provides access to the main functionality of the application.

Stocks

The Stocks tab allows user to analyze Stock symbols.

A great place to start is with our State Modeling in the Stocks menu. After clicking on State Modeling, you will be prompted to input a stock symbol.

The result is detailed information on the underlying security based on our State Modeling ™ formula. Once you find a security to trade, go to the quick backtest to test your trade.

Options

The Options tab will give you information regarding options for particular stocks.

 

Earnings

The Earnings tab will give you information regarding earning for particular stocks.

BackTest

The BackTest tab allows individuals three ways to perform a backtest of the underlying symbol.

An overview of the Quick BackTest and Wizard is provided here as these are most frequently used options of the platform.

1. Quick BackTest

The Quick BackTest panel is shown below.

<insert graphic>

Description of fields

  Symbol Mandatory field.

It is a text field to input the underlying symbol.

  Expiration Default set to Monthly. The field has 2 options – weekly or monthly expiration.
  Risk per Trade Mandatory field. The amount the user is willing to risk per trade.
  Strike Default is set to ATM. This field has 3 options: ITM (In the Money), ATM (At the money), and OTM (Out of the Money).
  Strategy Mandatory field. This field provides the list of options strategies that can be used.
  Days to Expire Mandatory Field. The number of days from expiration of the option contract.
  Profit Target (%) Mandatory Field. Exit position based on percent of profit made.
  State Mandatory Field. Delphian Proprietary State Modeling™ lets you choose from state 1 through 8.
  Stop Loss (%) Mandatory Field. Provide a specified price in order to limit a loss.

Execute a Quick BackTest

To execute the Quick BackTest:

  1. Input data into each field of the Quick BackTest panel.

  2. Click to execute the test.

    <insert graphic>

  3. When the test is successfully executed, the < g> button appears next to the <g> button. Click the <g> button to view the results.

2. Wizards

The Wizards tab will walk you through building a backtest. We have 8 different strategies to perform a backtest. Once you select a backtest Strategy, you will be prompted to give criteria to perform backtest.

Organizer

The Organizer tab allows the user to create a backtest by filing out and saving various criteria. By saving each criteria individually, you can use each criteria in future models. Once criteria has been created, go to the BackTest Reports and fill in Entry Exit dates, Trade rules and Group to perform a backtest.

Brokerage Links

The Brokerage tab will take you to your brokerage account.
