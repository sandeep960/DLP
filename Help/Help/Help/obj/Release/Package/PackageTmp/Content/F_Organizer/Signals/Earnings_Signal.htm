Earnings Signal

This filters for symbols that have a specific type of trading active during earnings periods.

 

Examples:

Mover: To filter symbols that are Mover candidates: select Field=Close, Days=1, Timing=After, Operator=Either Direction (Greater), Percent=5 and Earnings=3 out of 4.

 

Bullish: To filter symbols that are Bullish Runner candidates: select Field=Close, Days=20, Timing=After, Operator=”>=”, Percent=5 and Earnings=4 out of 4.

 

Bearish Runner: To filter symbols that are Bearish Runner candidates: select Field=Close, Days=20, Timing=After, Operator=”<=”, Percent=5 and Earnings=4 out of 4.

 

Bullish Pre-Runner: To filter symbols that are Bullish Pre-Runner candidates: select Field=Close, Days=20, Timing=Before, Operator=“>=”, Percent=5 and Earnings=4 out of 4.

 

Bearish Pre-Runner: To filter symbols that are Bearish Pre-Runner candidates: select Field=Close, Days=20, Timing=Before, Operator=“<=”, Percent=5 and Earnings=4 out of 4.

 

Volatility Rise: To filter symbols that are Volatility Rise candidates: select Field=ATM IV, Days=20, Timing=Before, Operator=“>=”, Percent=25 and Earnings=4 out of 4.

 

Volatility Crush: To filter symbols that are Volatility Crush candidates: select Field=ATM IV, Days=1, Timing=After, Operator=“<=”, Percent=35 and Earning=4 out of 4.
