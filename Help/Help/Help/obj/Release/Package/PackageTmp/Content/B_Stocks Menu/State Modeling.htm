State Modeling

The State Modeling page provides a detailed information on stock based on Delphian proprietary State Modeling™ algorithm.

Description of fields

 

Symbol

It is a text field to input the underlying symbol.
 

Stock List

Select the option as Stock List and then open the drop-down to select from the pre-defined or user-defined stock list.
 

My WatchList

Select the option as My WatchList, the results for the stocks previously entered in the My WatchList list are displayed.
 

Today’s Transitions

Select the option as Today’s Transitions and select from the available States in the drop-down.
 

Clicking this directs you to the Earnings> Analysis page where the page provides an in-depth earnings analysis for the selected stock symbol.
