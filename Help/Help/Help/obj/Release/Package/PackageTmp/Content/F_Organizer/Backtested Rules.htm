Backtested Rules

The Backtested Rules page displays all the backtested trade rules.

 

Editing trade rules for backtest

  1. Click the Trade Rule to be edited. This directs you to the trade rules page of the backtest.
  2. Make changes to the trade rules and click to override and change the rules

Click to create a copy of the same rules. Provide a different name for the rules and click to save the changes.
