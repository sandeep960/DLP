BACKTEST

The BackTest tab gives the user the ability to create various stock and option backtests in the platform.  The user has the ability to run backtests using the simplified version in Quick BackTests or by starting from ground using the backtest wizard.

 

Categories under BackTest include:

 

BackTest Reports

The user can view results of previous backtests.

 
 

Quick BackTest

The user can create a quick backtest based on State Modeling.

 
 

Wizard

The wizard guides you to create 8 different types of backtests.
