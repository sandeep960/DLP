Entry-Exit Dates

The Entry-Exit Dates page lists all the entry-exit dates.

Description of fields on this page:

Click this icon to expand the window with the list of trades for the symbols defined in the list.

 

Click this icon to display the criteria for the entry-exit dates. This icon is not displayed for manually created entry-exit dates.

Adding new Entry-Exit Dates

To add a new Entry-Exit Dates, click .

The Entry-Exit Dates page has 3 options to change Entry-Exit dates. By Default, the Manual mode is selected.

  1. Manual
  2. Buy Sell Criteria
  3. Import

Creating Entry-Exit dates in Manual Trade Mode

In Manual Trade mode, the user has to manually create and update all the fields and then execute a BackTest to view the results.

 

Steps to update Entry-Exit Dates in Manual Trade mode:

  1. Input a name in the Entry-Exit Dates field
  1. Input a Symbol, Entry Date and Exit Date.
  2. Click to add more rows to input symbols.
  3. Click to create the trade.

 

Creating Entry-Exit Dates in Buy-Sell Criteria Mode

In the Buy-Sell Criteria mode, the user selects the entry-exit dates based on criteria that include:

  • Earnings  -  by earning dates
  • State Model – by states
  • IV Ranker – by IV ranks
  • Calendar – by monthly or weekly calendar dates
  • Signal – by predefined signals

This page has the following fields:

Entry-Exit Dates

Entry-Exit Dates

Enter a name for the Entry-Exit dates selected.

 

Symbol or Stock List

By default, the Symbol tab is selected. Enter a symbol in the symbol field.

The StockList tab is only available for State Model, IV Ranker, Calendar and Signal criteria.

In the StockList tab, a drop-down menu is provided with Pre-Defined or User Defined lists.

NOTE: User-Defined lists can be created from the Stock List option in the Create menu.

 

 

Symbol to Trade

This field appears only with the Symbol tab. If you enter a symbol in this field, then the criteria is applied to the market or stock entered in the Symbol field but trading happens to the symbol in this field.

Example: Symbol entered is NDX with bullish criteria and the Symbol to Trade is AAPL, and then AAPL will be traded only if NDX is in bullish state.

 

Allow Simultaneous

By Default, the No option is selected.

This option is only available for Earnings, IV Ranker, Calendar and Signal criteria.

The Yes option lets the user enter the number of simultaneous active trades at the same time.

State 1, State 3, State 5 or State 7 for a bullish trend and State 2, State 4, State 6 or State 8 for a bearish trend.

Steps to create Entry-Exit Dates:

  1. Provide a name for Entry-Exit Dates.
  2. Input a symbol or select from the stock list.
  3. Select Calendar as criteria.
  4. Select From Date, To Date and the Monthly (or weekly) option.
  5. Select the entry and exit mode, input the number of days in each mode and number of months to skip.
  1. Click to create the Entry-Exit dates criteria.

 

Signals

In this option, signals are created with State Model values and are used to update the entry-exit dates based on these signals.

Two signals are created in this criteria - one for entry condition and the other for exit condition. By doing this, the user will be able to generate a report based on their own criteria for entering and exiting when the states transition from one state to another.

Creating Entry-Exit dates in Import Trade Mode

In the Import Trade mode, the user can upload a Microsoft Excel sheet with updated values and execute the result.

Steps to update Entry-Exit Dates in Import Trade mode:

  1. Enter a name for the Entry-Exit Dates field.
  2. Click icon to download and see the required format of the sheet.

  1. In the Excel sheet, enter the entry date, exit date and the symbol.

Save the excel sheet after updating.

  1. Upload the updated excel sheet by using the option provided as shown:

  1. The application checks the accuracy of the data and provides a preview of the data entered in the excel sheet.

  1. Click the button to import the data from the sheet into the application.
