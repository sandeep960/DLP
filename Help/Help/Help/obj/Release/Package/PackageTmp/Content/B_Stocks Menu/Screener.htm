Screener

The Screener page lists all the different signals that can be used to filter stocks and options.

There are 2 types of Screener options:

  1. Pre-Defined This displays a list of signals created by Delphian which the user can select.
  2. Custom This lets the user select the user provided signals and provides the values to each of the selected signals.

NOTE: The Signals listed on the screener page are described in the Signals section.

Pre-Defined Screeners

Example: Selecting stocks with Pre-Defined Earnings criteria
  1. Expand Earnings and select Pre-Runner.

(a) Expanding Earnings provides the list of criteria below it.

(b) The numbers listed next to each criteria are the number of stocks in that particular criteria.

(c)  The selected criteria is displayed as the result heading and selected multiple criteria will display all the selected criteria with their results.

  1. You can add the results to Stock List or Daily Watch list.

 

Adding result to Stock List

(a) Click on the results page:

(b) A window will open with the option to create a new stock list or override an existing list, as shown below. Input a name if you want to create a new list or select from existing lists.

Adding result to Daily Watch

To add result to Daily Watch:

(a) Click on the results page.

(b) A window will open with the option to input the name for the Daily Watch is seen. Provide a name for the Daily Watch criteria.

Custom Screeners

Example: Selecting stocks with Asset=ETF, Price (current) from $10 to $40 and Price trend increasing for the past 2 days

1. To select an Asset signal:

(a) Click Asset, a pop-up window with Asset filter opens.

(b) Select Asset as ETF

2. To Select Price (current) from $10 to $40:

(a) Click Price, a window with Price filter opens.

(b) Select Price=Current, Min=10 and Max=40.

 

3. To Select Price with Trend increasing for the past 2 days:

(a) Click Price, a window with Price filter opens.

(b) Select Price=Trend, Direction=Increasing, Days=2.

(c) Click Execute to display result.

4. Remove ETF criteria:

In case you want to remove the ETF criteria from the results above, click the xnext to this criteria and execute the result again.

The result without the ETF criteria will then be displayed.
