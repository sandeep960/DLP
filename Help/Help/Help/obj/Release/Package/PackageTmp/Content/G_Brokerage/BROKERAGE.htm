BROKERAGE

The Brokerage tab lets the user create an order and trade on the Delphian application. The user needs to register with Tradier Brokerage to be able to start placing orders and trading. Tradier Brokerage offers trading with no commission or trading fee.

Brokerage is only active and all items displayed when the user has registered an account with Tradier brokerage.

 

Categories under Brokerage include:

  Account Summary This provides general account information like cash balances, buying power and type of account.
  Positions

This page displays the open, pending, closed and link positions of the trade.

 
  Orders

The Orders page displays the created orders under processing.

 
  Live Data This displays live trading data of the symbol.

 
  Create Order This page lets the user create their trading orders from this page.
