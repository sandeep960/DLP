State Model Wizard

State Modeling Backtest is designed to help you design a strategy using State Modeling.

Use state modeling to determine whether the underlying is in a bullish or bearish pattern. Backtest different options strategies to find out which produces results based on your trading style.

 

In this section, we are going to show you how to perform a BackTest in the State Model Wizard by using an example:

  • We are going to BackTest a Bullish strategy for AAPL. 
  • We are going to purchase one long call every time AAPL has entered into State 1 since 0103/2007.
  • Profit target: 25% of the Initial Cost
  • Strike Selection: 45-55 Delta Range
  • Days to expiration: minimum of 60 days to maximum of 100 days

 

There are 3 steps to complete this BackTest: Entry-Exit Dates >> Trade Rules >> Preview

Entry-Exit Dates:

  1. Input a name for the dates to be saved under Test AAPL. 
  2. Input the stock symbol as AAPL. 
  3. Input the date range to be tested in From Date 01/03/2007 and To Date as 05/01/2017.
  4. Select the state you want to test as State_1.
  5. Click to save your Entry-Exit Dates and to move on to Trade Rules.

Trade Rules:

  1. Input a name for the trade rules to be saved under Long Call AAPL. 
  2. Click the Strategy drop down menu and select Long Call.
  3. Input the minimum and maximum days to expiration as 60 and 100.
  4. Input the minimum and maximum delta range 45 and 55.
  5. Input the profit target % as 25 based off the InitialCost.
  6. Input the stop loss % as 100 based off the InitialCost.
  7. Click to save your Trade Rules and to move on to Preview.

Preview:

All fields will auto-populate.  For risk per trade, you can select a dollar amount/trade, contracts/trade or a custom amount created manually.

  1. Click to have the system run the tests.
  2. Select to go to the BackTest results page.
