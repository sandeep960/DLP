State Model Signal

This signal is created based on State Modeling™ technique. It is a proprietary signal from Delphian Trading.

Stocks are in a State 1 to State 8. Odd numbers are bullish and even numbers are bearish. State 1 is extremely bullish while State 8 is extremely bearish.

See the State Modeling PDF to learn how the State Model works.

 

Example:

To find symbols that are in current bullish State 1 when buying and in Bearish state 6 when selling: select Type=Current then select “State 1” in the second column and “State 6” in the third column.
