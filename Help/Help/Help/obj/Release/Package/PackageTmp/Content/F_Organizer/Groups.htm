Groups

Groups are file cabinet for similar trade plans.

The Groups page displays the list of groups.

Clicking displays all the trades in that group.

Adding a new group

Click create a new Group.

 

Description of fields:

 

Group

Group Name

Mandatory

It is a text-field to input a new group name.

 
 

 

Context

Mandatory

It is a drop-down menu with the following options:

  • Earnings
  • Trend
  • Manual
  • Runner
  • Hedge
 

 

Description

Optional

It is text-field to provide a description of the group.
 

Click this button to create the new group.
 

Click this button to view previously created groups.

 

Example: Creating a group with 2 different long calls

  1. In the Organizer>Trade Rule page, create a long call strategy with delta range between 40 and 60.
  2. Create another long call strategy with delta range between 25 and 75.
  3. Add symbols to trade from the Organizer>Entry Exit Dates.
  4. Create a group from on the Organizer>Groups page.
  5. Go to the BackTest>BackTest Reports and run the backtest for the long call with delta range 40 to 60 and select group as Groups-longcall.
  6. Repeat the same by running a backtest for long call with delta range 25 to 75 and select group as Groups-longcall.
  7. You can view the created group along with the results in Organizer>Groups page.
