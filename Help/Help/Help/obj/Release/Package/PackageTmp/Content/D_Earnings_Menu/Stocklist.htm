Stock List

The Stock List page displays the Earnings Analysis of the selected Pre-Defined or User-Defined Stock List.

 

Pre-Defined Stock Lists are created by Delphian Trading in the application.

User-Defined Stock List can be created from the Organizer menu, under Stock List.

  1. In the example below, we created a list "Triple A" with Amazon, Inc. (symbol AMZN), Apple Inc. (symbol AAPL) and American Airlines Group Inc. (symbol AAL).
  2. Select "Triple A" as the Stock List and the earnings analysis is displayed for the symbols in the list. Use the to navigate from one symbol result to another.
