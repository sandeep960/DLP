Sentiment

The Sentiment option displays the Put/Call Ratio, Put/Call Options Volume and Put/Call Open Interest of the symbol.

Description of fields

Sentiment

Symbol

It is a text field to input the underlying symbol.
 

Put/Call Ratio Chart

The put-call ratio is used to gauge the overall sentiment (mood) of the market. The ratio is calculated by dividing the number of traded put options by the number of traded call options. A reading above 1 indicates put volume exceeds call volume. A reading below 1 indicates call volume exceeds put volume.

 

Put/Call Options Volume

The Put/Call Options Volume ratio is used to gauge market sentiment. It is calculated by taking the number of put options traded divided by the total number of options traded. A reading above 50 indicates put volume exceeds call volume. A reading below 50 indicates call volume exceeds put volume.

 

Put/Call Open Interest

The Put/Call Open Interest ratio is used to gauge market sentiment. The Put-Call Ratio is the number of put options traded divided by the total number of options traded using open interest volume. A reading above 50 indicates put volume exceeds call volume. A reading below 50 indicates call volume exceeds put volume.
