OPTIONS

The Options tab allows the user to access information on options of underlying securities.

 

Categories under Options include:

 

Option Chain

The user can view option quotes for the underlying security on all available expiration.

 
 

Sentiment

You can review sentiment indicators, Put Call/Ratio, Put/call volume and Put/Call Open Interest.
 

Volatility

This provides Volatility Skew and Implied Volatility chart.

 
 

Unusual Options

This provides chart of option volume and open interest compared to its 20 day average.
