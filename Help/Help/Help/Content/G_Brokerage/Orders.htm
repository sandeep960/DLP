Editing Pending or Open orders

If the order is created in the Delphian Trading, you can click to edit pending orders.

If the order is created in Tradier (Manual) you will see message “Cannot edit the trade from this platform which are opened directly in brokerage UI.”

To edit an order:

User can only edit duration, order type and price before resubmitting the order.

  1. Edit duration, order type or price of the order.
  2. Click Re-Submit Order to validate and update the order in both Delphian Trading and Tradier.
  3. If validation fails then error message response is received from Tradier.

Canceling Pending or Open orders:

An order created in Tradier can be canceled by clicking .

To cancel an order:

  1. Orders created in Delphian Tradingcan be cancel by clicking Cancel Orderfrom the preview page or click to cancel an order created in Tradier.

  2. A message is displayed for confirming the cancellation.

Creating new order from canceled/rejected/expired orders

  1. The canceled, rejected and expired orders pages have the button from where the user can preview and edit duration, order and price.
  2. Click Preview to see the commission, cost and fees for the order. This page also displays a Submit Order button if the order data is correct, else an error message is received from Tradier.
  3. Click Submit Order to update order data and leg and create a new order in Tradier. A new order number is displayed after creating the order successfully.
