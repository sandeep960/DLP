Create Order

The Create Order page lets the user create their trading orders from this page.

Creating an order

  1. Input a symbol. This displays the price, ask, bid and volume details of the symbol.
  2. Select a Strategy and Risk Per Trade from the list. .

This displays the following page:

Description of fields:

  Symbol Mandatory It is a text-field to input stock symbol.
  Strategy   A drop down menu with list of strategies.
  Risk Per Trade   A drop down menu with list of Risk Per Trade. See Risk Per Trade for details.
  Duration   Duration can be Good-Til-Cancelled (GTC) order that lasts until the order is completed or canceled or a day order that lasts until the end of trading day.
  Brokerage Mandatory Name of the brokerage, in this case Tradier.
  Accounts   Account number of the brokerage account.
  Order Type  

This has 2 choices:

  • Market - A market order is a buy or sell order to be executed immediately at current market prices. As long as there are willing sellers and buyers, market orders are filled.
  • Limit - A limit order is an order to buy a security at no more than a specific price, or to sell a security at no less than a specific price.
  Spread   The Buy-Sell Spread is the difference between the prices quoted.
  Price   The quoted price to trade the symbol.
 

 

OPTION LEGS

   
  Entry Date   Date when the order is created.
  Type   BTO by default. BTO is a Buy to Open that represents the opening of a long call or put positions. A buy to open order has a distinguishing characteristic where the option position is not held short in the account during the transaction. The sell to close(STO) order is used to exit a position taken with a buy to open order.
  Leg   It is the trading contract.
  Option Type   There are many different types of options that can be traded and these can be categorized in a number of ways. In a very broad sense, there are two main types: calls and puts. Calls give the buyer the right to buy the underlying asset, while puts give the buyer the right to sell the underlying asset.
  Expiry Date   It is a drop down menu to select the expiry date of the order.
  Strike   It is a drop down menu to select the strike price of the order.
  Price   Automatically populated after selecting strike price.
  Bid   The minimum bid price of the symbol.
  Ask   The maximum bid price of the symbol.
  Cnts   It is a text-field to input number of contracts you want to purchase.
  Option Symbol  

This displays the Option Symbol number.

 

3. Select the Price, Expiry Date, Strike and Count.

4. Click to see the order details and submit.
