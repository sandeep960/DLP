BackTest Report

BackTest

The BackTest Report page lets the user run tests with the manually created Entry-Exit Dates, Trade rules and Groups from the Organizer menu.

Previously executed backtests are also listed on this page.

 

 

BackTest

Provides a list of previously run Backtests.  Users can populate previous runs or manual selecting Enter-Exit Dates, Trade Rules & Group from their respective drop-down menus.
 

Group

User can view results of previous group reports by selecting the Enter-Exit Dates and Group from their respective drop down menus.
 

Run

User can view results of previous run reports by selecting the Enter-Exit Dates, Group & Run from their respective drop down menus.
 

Trade

User can view results of previous trade reports by selecting the Enter-Exit Dates, Group & Trade ID from their respective drop down menus.
 

View Trade

User can view results of a specific trade report by selecting the Enter-Exit Dates, Group, Run and Trade from their respective drop down menus.

 

Steps to execute a BackTest Report:

  1. Select the Entry-Exit Dates from the list.

The icon appears next to the Entry-Exit Dates field when you make a selection.

Clicking on this displays the entry-exit date criteria:

  1. Similarly, select the Trade Rules and the Group.

Selecting Trade Rules displays the following additional fields in the panel:

  • $ per TradeInput the dollar amount for the trade.
  • ContractsInput the number of contracts for the trade.
  • CustomThis is selected by default. Select the Risk Per Trade from the drop-down list.

  1. Click BackTest to execute the BackTest:

  1. The result of the BackTest can be seen by either clicking or the icon from the Previous BackTests panel.
Groups

The Group page displays the results of the BackTest and Quick BackTests.

Steps to display Group results:

  1. Select an Entry-Exit Date from the drop-down list.
  1. Select a Group.
  1. The Group results are displayed.

The Group page has the following fields:

Group

The Group panel can be used to select pre-defined or user-defined fields for Entry-Exit Dates and Groups. A description can be provided for the executed Quick BackTest in the description field.

 

Summary

The Summary panel displays the Quick BackTest result.
