Capital Plan

Capital Plans are used to combine 2 different strategies and see the risk reward reports using the run format.

The Capital Plan page displays the capitals plans.

 

Clicking displays the groups under the capital plan.

To move a group from one plan to another, select the group and click button to move.

Example: Creating a capital plan by combining long call and long put strategies

  1. In the Organizer>Trade Rule page, create a long call strategy.
  2. Create another strategy with long put.
  3. Add symbols to trade from the Organizer>Entry Exit Dates.
  4. Create a group longcall-cp from on the Organizer>Groups page.
  5. Create another group longput-cp.
  6. Create a capital plan from on the Organizer>Capital Plans page.

  7. Go to the BackTest>BackTest Reports and run the backtest for long call and select group as longcall-cp. After the result is executed, select the run on the group tab and click to add it to the capital plan.

  8. Repeat the same by running a backtest for long put and select group as longput-cp. Repeat the same as in above step by selecting the run on the groups tab and adding it into the same Capital Plan.
  9. You can view the results on the Organizer>Capital Plans page.
