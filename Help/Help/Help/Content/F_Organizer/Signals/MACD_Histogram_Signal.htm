MACD Histogram Signal

As with MACD, the MACD-Histogram is also designed to identify convergence, divergence and crossovers. The MACD-Histogram measures the distance between main line and the signal line. The histogram is positive when the main line is above the signal line. The indicator is negative when the main line is below the signal line.

 

Example:

Current:

If you want to find symbols where the MACD Histogram crossed below zero: select MACD Histogram=Current, Operator=Cross Below and Value=0.

 

Trend:

If you want to find symbols with their MACD Histogram in a downtrend for the past 20 days: select MACD Histogram=Trend, Operator=Decreasing and Days=20.

 

Consecutive:

If you want to find symbols with their MACD Histogram increasing every day for the past 5 days: select MACD Histogram=Consecutive, Direction=Increasing and Days=30.
