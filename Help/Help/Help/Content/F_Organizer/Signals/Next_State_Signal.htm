Next State Signal

This signal is created based on State Modelling™ technique. It allows users to find symbols based on their current state and their next highest probable state.

 

Example:

To find symbols that are in State 2 with their next highest probable state of State 1: select Current State=State 2 and Next State (High Probability)=State 1.
