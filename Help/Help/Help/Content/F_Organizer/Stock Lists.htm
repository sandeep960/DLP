Stock Lists  

The Stock Lists page displays the user defined and pre-defined stock lists.

 

 

Editing a user defined stock list

Click a stock name to open a page that displays the symbols in the list and to edit the list.

Steps to edit the list:

  1. In the Stock List symbol page, input a symbol in the Symbol field or use the search option to search for a specific symbol from the list and select the symbol from the search results.
  2. Click to update the list.

To move a symbol from one list to another:

  1. In the Stock List Symbol page, select the symbols to move.
  1. Click to open the stock lists and select the stock to move the symbol.
  1. Click to update the lists.

 

Adding a new Stock List

On the Stock List page, click to create a new Stock List.

The Stock List page lets the user create their own list of stocks. Stocks are created in the following 3 ways:

  1. Manual
  2. Import
  3. Signal

Creating a Stock List in Manual mode

 

In the Manual Stock List, the user provides the stock list name and selects the stocks from the list of stocks provided below.

 

Steps to create Manual Stock List:

  1. Provide a name in the Stock List Name field.

  2. The Symbol field can be updated in the following ways:

    • Select the symbols from the list below and they will be automatically updated in the Symbol field. The multiple pages can be navigated through left and right arrows or by clicking on the page numbers.
    • In the Search option, provide the name of the symbol to display the list of stocks that match the search.
    • In the Symbol field, provide the first few characters of the symbol to display the list of symbols matching those characters and then select the desired symbol.
  3. Click to create the Stock List.
Creating a Stock List in Import mode

In the Import Stock List, the user imports a Stock List from Microsoft Excel sheet.

The Import Stock List lets the user upload the list of stocks from a Microsoft Excel sheet. The Excel sheet is in a pre-defined format that has to be downloaded, saved and then updated with the symbols.

  1. Provide a name in the Stock List Name field.
  2. Click the icon to open the Microsoft Excel sheet.
  1. Update the excel sheet with list of symbols.
  1. Save and upload the files.
  1. When the excel sheet is uploaded a window with the list of symbols is displayed.
  1. Click to create the Stock List.

Below are examples with steps to create Signal stock lists with the specified criteria.

Example 1: A single new stock list with State Model criteria
  1. Select Symbol tab and input a symbol.
  1. Select New StockList and input a name for the list.
  1. Select State Model criteria.
  1. In the State Model criteria window, select the type and state.
  1. Click to create the stock list.
  1. The result with the selected criteria is displayed.

 

Example 2: An existing Stock List with Override Existence and SMA criteria
  1. Select StockList tab and select a stock list from the previously created lists.
  2. Select Override Existence and select a list from the previously created lists that has to be overwritten with the list selected.
  3. Select SMA criteria. In the SMA criteria window, select the field, operator and input value for Days1.
  4. Click to create the stock list.
  1. The result with the selected criteria list is displayed.
Example 3: A new Stock List with Sectors criteria
  1. Select StockList tab and select a stock list from the previously created lists.
  2. Select New StockList. and input a name for the new stock list to be created.
  3. Select Sector criteria. In the Sector criteria window, select the sector and the industry.
  4. Click to create the stock list and see the result.
