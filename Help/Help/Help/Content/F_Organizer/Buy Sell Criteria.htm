Buy Sell Criteria

The Buy-Sell Criteria page displays the user defined and pre-defined buy sell criteria.

Clicking any of the criteria directs you to the page with the defined parameters of those criteria. You can edit and save them as needed.

Adding a new Buy Sell Criteria

To add a new Buy Sell Criteria, click .

The Buy Sell Criteria lets the user create their own criteria based on earnings, state modeling, IV ranker, calendar or signal.

The Buy Sell Criteria page is as shown:

The page has the following fields:

Buy-Sell Criteria

Buy-sell criteria

It is a text-field to input a name for the Buy-Sell criteria.

 

Description

It is an optional text-field to provide a description for the Buy-Sell criteria.

Buy-Sell Criteria

There are 5 criteria:

  • Earnings
  • State model
  • IV ranker
  • Calendar
  • Signal

Earnings

The earnings criteria can be created based on the following types:

  • Mover
  • Volume Riser
  • Non-Mover
  • Runner
  • Pre-Runner

Creating an Earnings Buy Sell Criteria

Steps to create Earnings Buy Sell Criteria:

  1. Input a Buy Sell Criteria name and description.
  1. Click Earnings criteria.
  1. The options with different types of earnings are displayed.
  1. Select any one of the option.
  1. Click to save the new criteria.

 

State Model

This criterion is based on State Modeling™ where a symbol is traded based on its state.

Additional fields in the State Model criteria page:

Buy-Sell Criteria

Symbol to Trade

It is a text-field to input the underlying symbol.

State-Model

From Date and To Date

Select a date range. If you click the icon in both fields, the date range is selected from 1/3/2007 to the current date.

 

 

State

Select a state from the following list:

  • All States
  • Bullish
  • Bearish
  • State_1
  • State_2
  • State_3
  • State_4
  • State_5
  • State_6
  • State_7
  • State_8
Creating State Model Buy Sell Criteria

Steps to create State Model Buy Sell Criteria:

  1. Input a Buy Sell Criteria name and description.
  1. Click State Model criteria.
  1. Input a symbol.
  1. Select the From Date and To date.
  1. Select State from the list.
  1. Click to save the new criteria.
 

IV Ranker

This criterion is based on Implied Volatility (IV) Rank.

Additional fields in the IV Rank criteria page:

Buy-Sell Criteria

Symbol to Trade

It is a text-field to input the underlying symbol.

 

 

Allow simultaneous

By default, the no option is selected.

The yes option lets the user input the number of simultaneous active trades at the same time.

State 1, state 2, state 3 or state 4 for a bullish trend and state 2, state 4, state 6 or state 8 for a bearish trend.

 

IV Ranker

From Date and To Date

Select a date range. If you click the icon in both fields, the date range is selected from 1/3/2007 to the current date.

 

 

IV Days

Select the days from the list of days.

 

 

Entry and Exit

 

 

Increase/Decrease      

Select the entry direction from the Increase or   Decrease option.

 

 

Entry and Exit Quartile

Select the Quartile from the list:

  • All Quartile
  • Q1
  • Q2
  • Q3
  • Q4
Creating an IV Rank Buy Sell Criteria

Steps to create IV Rank Buy Sell Criteria:

  1. Input a Buy-Sell Criteria name and description.
  1. Click IV Rank criteria.
  1. Input a symbol.
  1. Select yes or no for Allow Simultaneous.
  1. Select the From Date, To Date and IV Days from the list.
  1. Select the Increase or Decrease, Entry and Exit Quartile from the list.
  1. Click to save the new criteria.
 

Calendar

This criterion is based on monthly and yearly calendar days.

Additional fields in the Calendar criteria page:

Buy-Sell Criteria

Allow simultaneous

By default, the no option is selected.

 

The yes option lets the user input the number of simultaneous active trades at the same time.

State 1, state 2, state 3 or state 4 for a bullish trend and state 2, state 4, state 6 or state 8 for a bearish trend.

 

Calendar

From Date and To Date

Select a date range. If you click the icon in both fields, the date range is selected from 1/3/2007 to the current date.

 

 

Monthly/Weekly

Select any one of the option.

 

 

Monthly Entry and Exit

 

 

Entry or Exit      

3 options available:

  • Days of Month
  • Days after Monthly Expiry
  • Days before Monthly Expiry

 

 

Number of Days

It is a text-field to input number of trade days.

 

 

Skip Months

It is a text- field to input the number of months to be skipped.

 

 

Weekly Entry and Exit

 

 

Day of the week

It has the following options:

  • Pwd*-Thursday
  • Pwd-Friday
  • Monday
  • Tuesday
  • Wednesday
  • Thursday

(*  PWD – Previous Working Day)

 

 

Skip Weeks

It is a text-field to input number of weeks to be skipped.

 

Creating a Calendar Buy Sell criteria

Steps to create Calendar Buy Sell criteria:

  1. Input a Calendar criteria name and description.
  1. Click Calendar criteria.
  1. Select either yes or no for Allow Simultaneous.
  1. Select a From Date, To Date and monthly or weekly option.
  1. Select the Entry and Exit Day of the Week and Skip Weeks.
  1. Click to save the new criteria.

Signal

This criterion is based on signals created in the Create > Signals. See section Signal for steps to create signals.

Additional fields in the Signal criteria page:

Buy-Sell Criteria

Symbol to Trade

It is a text-field to input the underlying symbol.

 

Allow simultaneous

By default, the no option is selected.

 

The yes option lets the user input the number of simultaneous active trades at the same time.

State 1, state 2, state 3 or state 4 for a bullish trend and state 2, state 4, state 6 or state 8 for a bearish trend.

 

Entry and Exit

From Date and To Date

Select a date range. If you click the icon in both fields, the date range is selected from 1/3/2007 to the current date.

 

 

Signal

Select from the list of User or pre-defined Signals from the down-down window.

NOTE: By default, the list is sorted by date in descending order in all drop-down selection lists.

The sorting of the list can be changed by clicking up or down  icon on Date or Name. You can also search an item by inputting the name in the Search field.

When you select a Signal, you can see an Indicator panel that shows the parameters that are used to create the Signal.

Creating a Signal Buy Sell criteria

Steps to create Signal Buy Sell criteria:

  1. Input a Signal criteria name and description.
  1. Click Signal criteria.
  1. Select yes or no for Allow Simultaneous.
  1. Input a Symbol to Trade.
  1. Select a From Date and a To Date.
  1. Select the entry and exit Signal.
  1. Click to save the new criteria.
