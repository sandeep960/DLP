ORGANIZER

The Organizer tab stores all the values and parameters to be used for State Modelling analysis and BackTesting.

 

Categories under Organizer include:

 

Risk Per Trade

The Risk Per Trade page lets the user define the amount the user is willing to invest in trades.

 
 

Signals

Signals are used to create a filter based on certain criteria for symbols to be traded.

 
 

Stock Lists

The Stock Lists page displays the user defined and pre-defined stock lists. User can to create their own Stock Lists from Create.

 
  Buy Sell Criteria

The Buy Sell Criteria page displays the user defined and pre-defined buy sell criteria. User can create their own Buy-Sell criteria based on earnings, state modeling, IV ranker, calendar or signal.

 
  Entry-Exit Dates

The Entry-Exit Dates page list the defined entry and exit dates for the trades. User can create their entry-exit dates either manually, based on Buy Sell criteria or import from an excel sheet.

 
  Trade Rules Trades Rules page helps user create their own set of values for each trading parameters.

 
  Groups

The Groups page lets user create groups for similar trade plans.

 
  Capital Plans

The Capital Planspage lets the user combine 2 different strategies and see the risk reward reports using the run format.

 
  Backtested Rules

The Backtested Rules page lists all the BackTests executed in the BackTest>Quick BackTest page.

 
  Daily Alerts

The Daily Alerts page lets the user see the status and criteria of the daily alerts that are added from the run results of the BackTest>Quick BackTest page.
