Analysis

The Analysis page provides in-depth information on the last 16 earnings reports for the selected stock. 

 

 

Description of fields

 

Criteria

Symbol

It is a text field to input the underlying symbol.

 

Stock List

It opens a window with the list of pre-defined and User defined stock list.

 

This directs the user to the Organizer>StockList page where it displays the pre-defined and User defined stock lists.

 

Fields displayed after selecting a symbol or stock list:

 

This directs the user to the State Analysis page of the selected symbol or stock list. See section State Modeling for more information.

 

This directs the user to the Stock Chart of the selected symbol or stock list.

 

This directs the user to the Option Chart of the selected symbol or stock list.

 

stock name

The name of the selected symbol.

 

Sector

This field displays the sector to which the symbol belongs.

 

Industry

This field displays the industry to which the symbol belongs.

 

Next Earnings

This field provides the next earnings date.

 

Close Price

This field provides the closing price of the company.
