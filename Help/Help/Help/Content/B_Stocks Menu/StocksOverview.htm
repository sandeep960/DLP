Stocks Overview

The Stocks Overview page provides multiple options to analyze an underlying stock.

Description of fields

SELECT

Symbol

It is a text field to input the underlying symbol.

 
 

Stock List

Select the option as Stock List and then open the drop-down to select from the pre-defined or user-defined stock list.

 
 

My WatchList

Select the option as My WatchList, the results for the stocks in the My WatchList list are displayed.

 
 

Today’s Transitions

Select the option as Today’s Transitions and select the State from the drop-down:

 
TABS

State

Provides detailed information for stocks based on Delphian proprietary algorithm.
  Profile Provides general information for the selected symbol.
 

Chart

Displays a detailed chart for the selected security.
 

Volatility

Graphically shows the current volatility skew and provides a historical chart with the implied volatility measures for 30, 60, 90, 120, 180 & 360 days.
 

Option chain

Displays the pertinent options data for all available expiration in the underlying security.
 

Financials

Provides information on the company’s Income Statement, Balance Sheet and Cash Flow based off of its recent SEC filings.
 

Key Statistics

Shows relevant valuation measures and statistics for the selected stock.
  Insider Trading

 

 		Provides a list of insiders and major stock holders and their recent transactions in the underlying stock.
  Earnings Displays an in-depth analysis centered on the past 16 earnings reports for the selected stock.
