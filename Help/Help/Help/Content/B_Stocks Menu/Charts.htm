Charts

The Charts option displays OHLC (open-high-low-close) line chart of the inputted symbol or list of symbols.

 

Input a symbol in the Symbol field or select a list from the Stock List drop-down list.

Load Default and Default Template

  • You can click theLoad Default icon to clear all the default values from the Symbol or Stock List chart.
  • The Default icon saves the newly created template as default for all charts.

 

Chart results notation

The default date range in the symbol chart results are from January 3, 2007 to previous trading day. 

You can change the date range to display only 1 month, 3 month, 6 month, Year to date (YTD), 1 year or the default date range All.

The icon on the top right of the chart expands to provide options to save the chart in PNG, JPEG, PDF or SVG vector image.

When you have one or more values being displayed in the line chart, you can click on any of the values to hide or display.
